Lecteur de Diaporama

Description :

Le but de ce projet était de créer pas à pas un lecteur de diaporama qui fonctionne avec des images (gif) stockés sur la machine et avec l'applications reliée à une base de données locale pour avoir les informations des diaporamas. Le projet était par groupe de 3, il était à faire sur Qt Creator en C++, puis implémenter une base de données contenant les informations des diaporamas.
